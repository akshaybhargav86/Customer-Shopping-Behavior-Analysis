# Customer Shopping Behavior Analysis

## Overview
This project performs Exploratory Data Analysis (EDA) on a customer shopping behavior dataset. The goal is to understand purchasing patterns, customer demographics, and revenue trends to generate actionable business insights.

---

## Dataset
The dataset includes:
- Customer demographics (Age, Gender, Country, State)
- Product details (Category, Sub-category, Product)
- Transaction data (Quantity, Cost, Revenue, Profit)
- Date of purchase

---

## Objectives
- Analyze customer purchasing behavior
- Identify top-performing product categories
- Evaluate revenue and profit trends
- Perform time-based analysis (monthly trends)
- Handle missing values effectively

---

## Data Cleaning
- Converted Date column to datetime format
- Extracted Month, Year for analysis
- Handled missing values:
  - Numerical columns → mean/median
  - Categorical columns → mode / "Unknown"
- Removed inconsistencies and ensured data quality

---

## Exploratory Data Analysis (EDA)
- Revenue and Profit analysis by category
- Customer segmentation based on Age and Gender
- Monthly sales trend analysis
- Top products and sub-categories
- Profit margin insights
